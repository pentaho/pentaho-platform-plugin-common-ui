Accordion Wizard

accordionWizard

Directives:
  • accordion-wizard - Main container
  • accordion-wizard-heading - Optional custom heading html. Either use this or the heading attribute on the accordion-wizard-group
  • accordion-wizard-group - Panel definition in the accordion wizard
