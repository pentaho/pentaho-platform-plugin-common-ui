Example usage of angular-directives

All directives have a few common requirements for use.
Be sure to use directives as attributes on DIVs rather than element names to support IE8 You need to make the javascript available to your application and then you will need to inject the desired module into your AngularJS app. Like this: 

            
                pen.require([
                'common-ui/angular',
                'common-ui/angular-ui-bootstrap',
                'common-ui/angular-directives/angular-directives'
                ], function(angular) {
                // code here
                });

                // inject the module that you want by name
                var app = angular.module('example', ['accordionWizard', 'folderBrowser', 'dateTimePicker']);
