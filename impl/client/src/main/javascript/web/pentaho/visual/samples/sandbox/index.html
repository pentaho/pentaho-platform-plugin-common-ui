This sample visualization container can be accessed in two ways:

  1. from within the Pentaho web server, e.g.: http://localhost:8080/pentaho/content/common-ui/resources/web/pentaho/visual/samples/sandbox/index.html, or,
  2. from another local web server, simulating an embedded scenario, and assuming that the Pentaho web server is available at http://localhost:8080/pentaho/.
